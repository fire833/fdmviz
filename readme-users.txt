# Instructions to Operate

1. Open the target URL (either `http://localhost:5173` or `https://fdmviz.tauser.us/`) in your (modern) browser of choice.

2. Select a desired file for visualizing (preset, upload, or link to an STL)

3. Select a View Mode to explore alternative implementations

4. Check features to use optional add-ons for each View Mode

5. In Simulation mode, step the simulation by hand or let it tick away automatically.

6. Pan and zoom the viewport to inspect different regions of the model.

7. Settings are saved on refresh so feel free to leave and come back later!
